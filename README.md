## 🌟 Overview
**JEYUK-FISHING SYSTEM** transforms the tedious vanilla fishing mechanics into a thrilling, high-stakes adventure. Stop the boring "click-and-wait" gameplay! This script introduces a high-quality GUI-based Mini-game, Rod Progression, and Mythical Collections that will keep your players hooked for hours.

### ⚙️ Technical Information
* **Tested Environment:** ✅ Paper MC 1.21.1+
* **Script Engine:** Skript v2.11.1+
* **Addons:** SkBee v3.11.2+ & Skript-reflect v2.6+
* **Optimization:** Designed for high-performance and low CPU usage.

---

## 📦 Dependencies
To ensure a smooth experience, your server must have the following:
1. **[Skript](https://github.com/SkriptLang/Skript/releases)** v2.11.1
2. **[SkBee](https://github.com/ShaneBee/SkBee/releases)** v3.11.2 (Required for GUI & Metadata)
3. **[Skript-reflect](https://github.com/Skript-Reflect/skript-reflect/releases)** v2.6 (Required for advanced functions)
4. **[Vault](https://github.com/MilkBowl/Vault)** v1.7.3 + **EssentialsX** (For Economy/Money integration)
5. **AuraSkills** (Optional: Integrated Fishing XP)

---

## 🚀 Core Features
* 🕹️ **Interactive "Reel-it-in" Mini-game:** Players must time their clicks on a moving bar within a custom GUI. Precision determines success!
* 📈 **Rod Tiering Progression:** Upgrade fishing rods from Tier 1 to Tier 5 (Mythic). Higher tiers unlock rarer fish and better rewards.
* 🐉 **Mythical & Secret Catches:** Rare encounters with legendary creatures like the **King Kraken**, **Leviathan**, and **Poseidon**, featuring epic lightning and particle effects.
* 💰 **Auto-Sell Toggle:** Players can choose to instantly sell their catches for money or keep them as physical items in their inventory.
* 🖼️ **Rare Collection Gallery:** A dedicated GUI menu to track which legendary fish a player has discovered and which are still missing.
* 🏆 **Optimized Cached Leaderboard:** Real-time Top 5 "Secret Fish" hunter rankings, designed to be lightweight and lag-free.
* ⚡ **Dynamic Visuals:** Includes custom sound effects, action bar notifications, and immersive particles for a premium feel.

## 🛠️ Commands & Description
| Command | Description |
| :--- | :--- |
| `/fish` | Opens the Main Fishing Menu (Auto-sell, Collection, Leaderboard) |
| `/fish upgrade` | Access the Rod Upgrade menu for the held item |

> *Note: Custom permissions have not been implemented yet, given the limited number of commands in this version.*

---

## 📥 Installation Guide
1. Download `JeyukFishing.sk`.
2. Upload the file to your server directory: `/plugins/Skript/scripts/`.
3. Run the command: `/sk reload JeyukFishing`.
4. Start fishing and enjoy the system!
